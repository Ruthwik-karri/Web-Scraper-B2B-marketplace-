B2B Marketplace Web Scraper / Data Scraper

This project collects product data from IndiaMART and performs exploratory data analysis.

*Features

- Web scraping using Selenium and BeautifulSoup
- Collects product data from multiple categories -(machinery,electronics,textiles)
- Saves structured data (JSON + CSV)
- Performs exploratory data analysis (EDA)
- Generates visualizations

*Categories Scraped

- Machinery
- Electronics
- Textiles

*Output

The script generates:

output/products.json  
output/products.csv

The script performs:

- Summary statistics
- Product frequency analysis
- Category distribution
- Visualization of top products
